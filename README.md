<h1 align="center">WIP: Isshin</h1>
<p align="center">Next.js starter kit</p>

## Getting Started

```bash
$ git clone https://github.com/Kingsley-Don/isshin.git
$ cd isshin
$ yarn
$ yarn run dev
```

## Features

- Next.js
- @babel
  - preset-env
  - polyfill
  - plugin-proposal-decorators
- postcss
  - preset-env
  - nested
- eslint + prettier
- husky + lint-staged
